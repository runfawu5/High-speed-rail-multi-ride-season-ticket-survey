# High-speed-rail-multi-ride-season-ticket-survey

This repository hosts the questionnaire file of the questionnaire used in the study:

**“Joint optimization of seat allocation and pricing with multi-ride season tickets in high-speed railway services”**  
(Working paper)

## Contents
- `Questionnaire for multi-ride season ticket in HSR.pdf` - PDF 

## Questionnaire Structure
- **A. Screening Questions**
- **B. Travel Frequency**
- **C. Factors Affecting Your Travel Frequency**
- **D. Stated Preference Choice Scenarios**
